# Data Analytics Projects Portfolio

## Overview
This repository contains data analytics projects completed using Tableau and Excel. The projects focus on analysing real-world datasets, identifying trends, and communicating insights through data visualisation and analytical reporting.

The visualisations created in Tableau are included within the analytical reports, and the packaged Tableau workbooks can also be downloaded to explore the dashboards interactively.

---

## Project 1: Australian Open Tennis Analysis

### Description
This project analyses historical Australian Open tennis data to identify trends in championship performance and player success over time. The analysis focuses on players with multiple titles and examines patterns in tournament outcomes.

### Tools Used
- Tableau
- Microsoft Excel

### Files
- **Australian_Open_Report.pdf** – Report containing visualisations and analysis.
- **Australian_Open_Dashboard.twbx** – Tableau packaged workbook with dashboards.
- **Australian_Open_Data.xlsx** – Dataset used for the analysis.

### Key Insights
- Identifies the most successful players in Australian Open history.
- Examines long-term trends in championship wins and runner-up performances.
- Uses visualisations to highlight dominant players and historical performance patterns.

The Tableau dashboards used in this project can be viewed within the report or downloaded to explore interactively.

## Project 2: Australian International Trade Analysis

### Description
This project analyses Australian trade data to explore trends in exports and imports across different commodity categories over time. The aim is to understand trade performance and identify major growth patterns.

### Tools Used
- Tableau
- Microsoft Excel

### Files
- **Australian_Trade_Report.pdf** – Report containing visualisations and analytical findings.
- **Australian_Trade_Dashboard.twbx** – Tableau packaged workbook with dashboards.
- **Australian_Trade_Data.xlsx** – Dataset used for the analysis.

### Key Insights
- Examines trends in Australian exports and imports over time.
- Identifies key commodity categories contributing to trade growth.
- Uses Tableau visualisations to highlight changes in trade patterns.

The Tableau dashboards are included in the report, and the packaged workbook can also be downloaded to explore the visualisations interactively.
